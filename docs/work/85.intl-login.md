# INTL 登录

## 登录核心流程

1. 授权登录之后，前端可以拿到 `INTLSDK` 的 `openid`、`token`，把这两个写入 `cookie` 中，`cookie` 名称分别为 `tip_intl_openid`、`tip_intl_token`。
2. `url` 中携带 `_ltype=tiploginintl&appid=xxx&channelid=xxx`，调用后台1个接口换取登录态，后台校验登录态通过之后，会在 `cookie` 中写入 `session_id`。
3. 后续调用接口不需要再携带 `_ltype=tiploginintl&appid=xxx&channelid=xxx`，只需要携带 `cookie` 中的 `session_id` 就可以了。
