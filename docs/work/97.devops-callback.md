# 蓝盾回调增删查

## 文档

1. https://iwiki.aow.com/p/1614669042

## 注意

1. devnet 回调

`devnet` 环境的回调会被自动转发，无需配置。如下面的 `http://idc.devops.oa.com/proxy-devnet`

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/11/own_mike_x53d4Adi82zzw3TQ.png" width="600" />

2. 创建的时候需要用 `encodeURIComponent` 包裹

```js
landun.createCallback(
  encodeURIComponent('xxx'),
  event,
)
```

3. 逻辑沉淀
  
增、删、查相关逻辑都放到[研发平台后台项目](https://git.aow.com/pmd-mobile/support/rd-platform-svr/blob/master/bin/project/devops-callback.js)了。
