# 人工审核配置化

在七彩石配置需要审核的项目、子工程、审核人。

例如：

```json
{
  "pmd-mobile/match/gp-next##project/gp": "young",
  "pvpesport/next##project/user": "lee,dog",
  "pvpesport/match/val-match": "lee",
  "pvpesport/match/*": "lee"
}
```

`key` 格式：

- 多个项目，`key` 为 `glob` 模式，如：`pmd-mobile/match/*`
- 单个项目所有子仓库，`key` 为 `projectName`
- 单个项目单个子仓库，`key` 为`${projectName}##${subProjectName}`，中间两个 `#`

多个审核人：

- 以 英文逗号`,`分隔
