# 加速 COS 图片上传

之前上传COS图片都是直接用的蓝盾插件，上传的整个目录，不管之前是不是已经上传了，也不管有没有修改，都重新上传一次。

随着文件越来越多，这种方式耗时越来越久，最近的几次达到了`两三个小时`。

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/9/own_mike_xZ4X33zK4TZz5a5n.png" width="600" />

优化了下，只上传修改的和新增的文件，现在只需要`1分多钟`，减少了 `99.9%` 耗时。

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/9/own_mike_p2WS55AyjkPBsSG6.png" width="600" />

对比下蓝盾插件和脚本：

|方式|易用性|灵活性|
|---|---|---|
|蓝盾插件|高|低|
|脚本|中|高|
