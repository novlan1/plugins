# 灰度发布操作指南

## 1. 灰度发布

1. 先在域名配置下开启白名单，`h5.igame.qq.com` 已开通，其他域名陆续开通中，如紧急体验，请联系 `novlan1`
2. 业务需引入简单的 `sdk`，进行 `cookie` 注入
3. 发布工程时，修改灰度比例，小于100时开启灰度发布，否则为正常发布

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/9/own_mike_DmFkM4JHaTdjPzms.png" width="600" />

## 2. 更新灰度比例

1. 研发平台子工程 => 发布记录 => 灰度发布 Tab 中，找到对应分支、环境，进行操作即可。
2. 三种操作类型有
   - 更新灰度比例
   - 灰度变全量
   - 撤销灰度
