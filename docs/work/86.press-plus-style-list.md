# Press Plus Style List 的说明

Style List 就是可供选择的多种游戏样式。

执行下面命令可以生成所有组件的 Style List 文件：

```bash
npm run gen:style
```

文件地址：[src/packages/common/config/all-style-list.json](https://git.aow.com/pmd-mobile/pmd-h5/press-plus/blob/release/src/packages/common/config/all-style-list.json)
