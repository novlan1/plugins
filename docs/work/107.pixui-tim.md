# PixUI 中 TIM 接入

PixUI 中支持接入 [TIM](https://web.sdk.qcloud.com/im/doc/v3/zh-cn/index.html)。

PUBGM 赛事中就用到了 TIM 的群组部分，主要是准备、取消准备给队友发消息。

流程如下：

1. 初始化 TIM 实例
2. 获取签名，TIM 登录，需在第一步实例化后进行
3. 初始化群组，这一步需要在 TIM `ready` 后进行，可将其放在 `oSdkReadyListener` 中，初始化步骤
   1. 构造 `groupId`，`groupName` 等信息
   2. 调用 `getGroupProfile` 看有没有存在这个 `groupId`
   3. 有就调用 `joinGroup` 加入，没有就调用 `createGroup` 创建
