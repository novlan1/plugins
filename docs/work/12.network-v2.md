# 网络框架 V2 版本

## 异步拦截器

支持异步拦截器。改动较大，保险起见，不在原地修改，新建了v2。

支持异步拦截器后，可以支持以下内容：

- 可以选择微信/QQ登录，等待用户选择，然后发起请求
- 其他业务场景，如异步处理请求参数
