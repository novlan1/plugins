# 海外项目伽利略上报

项目 `config.js` 下配置 `aegis` 的 `idV2` 和 `url`，如：

```js
module.exports = {
  // 其他配置...
  aegis: {
    // id，从伽利略平台获取
    idV2: 'SDK-xxx',
    // 上报 URL，与国内不同
    url: 'https://sg.galileotelemetry.tencent.com/collect',
  },
}
```
