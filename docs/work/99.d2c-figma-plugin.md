# D2C Figma Plugin

## 1. 插件地址

https://www.figma.com/community/plugin/1569401707734289442

## 2. 功能

1. 支持任意大小节点，小至一个button，大至一个页面，弹窗也支持（D2C网站只支持一个页面）
2. 支持下载代码、stackblitz 打开、复制代码

## 3. 插件选择

选择插件 => From Tencent => 游戏人生 D2C

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/11/own_mike_dCPk8nZi2McxaMxK.png" width="600" />

## 4. 使用方式

<img src="https://mike-1255355338.cos.ap-guangzhou.myqcloud.com/article/2025/11/own_mike_PmCd2GMeAP2paiBs.png" width="600" />

1. 选择节点
2. 简单校对（可选）
3. 点击“生成”
4. 复制代码、下载代码、stackblitz 打开

## 5. FAQ

### 5.1. 能否在插件中直接预览？

目前不可以。这种需要插件自己解析生成的组件，而我们的组件引用关系复杂，涉及多个底层组件库，比如 Press UI, Press Next 等，需要比较大的工作量。

而嵌入 stackblitz 的方式也有跨域问题，所以目前只能跳转到新的 stackblitz 页面中预览。
